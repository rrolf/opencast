X5gon User-Tracking Informationen

Auf dieser Seite ist die Software X5gon eingebunden. Mithilfe dieser Software sollen freie Bildungsressourcen (Open Educational Ressources - OER) gefunden und indiziert werden.

Zusätzlich sollen Nutzern basierend auf ihren Interessen, geeignete OERs vorgeschlagen werden können.

Um mit X5gon OERs zu suchen gehen sie bitte auf: https://platform.x5gon.org/search

Der auf dieser Seite eingebundene X5gon Code übermittelt folgende Daten bei ihrem Besuch:

  • userId: Mit diesem Attribut wird der Benutzer identifiziert der auf das OER Material zugreift. Die ID wird von der X5gon Plattform erstellt und hat das folgende Format: X. Ein typisches Beispiel würde folgendermaßen aussehen: 41731408528951275X1524236028520. Diese Form der ID stellt sicher das Niemand, auch nicht das X5gon Projekt den Benutzer direkt identifizieren kann. Diese Information ist aber ausreichend um einen Benutzer über eine Session hinweg erneut zu identifizieren um personalisierte Empfehlungen geben zu können.
  • materialURL: Dieser Wert ist eine Identifikation für das Material das der Benutzer besucht hat. Es wird dafür genutzt Inhalte für den Suchindex zu gewinnen und weiterhin auch für die Generierung von Empfehlungen und Learning Analytics. Die gewonnenen Inhalte werden angereichert und die angereicherten Metadaten werden in der X5gon Datenbank gespeichert. Die Quellen selber werden aber nicht gespeichert.
  • referrerURL: Diese URL ist der Link über den der Benutzer auf diese Seite gekommen ist. Diese Information wird analysiert um Muster im Lernverhalten des Benutzers zuerkennen und basierend auf dem Verhalten anderer Nutzer Empfehlungen geben zu können.
  • accessDate: Der Zeitpunkt zu dem auf die URL zugegriffen wurde. Diese Information wird dafür genutzt um ferstzustellen, ob der Nutzer seine Interessen geändert hat.
  • userAgents: Dieses Attribut enthät informationen darüber welche Technologie der Nutzer beim Zugriff auf das Material genutzt hat. Da auch viele Bots genutzt werden, wird diese Information benötigt um festzustellen, ob der Zugriff von einem normalen Nutzer oder einer Suchmaschine erfolgte und so fehlerhafte Daten auszusortieren.
  • location: Die IP-Adresse des Nutzers, die dafür genutzt wird den Ort festzustellen. Diese Information wird dafür benötigt das Benutzerprofil and kulturelle Eigenschaften anzupassen, die geografisch beeinflussed sein können. Die IP-Adresse wird nicht in der Datenbank gespeichert, im Gegensatz zu der ermittelten Region.

Wenn sie eine Auskunft zu ihren Daten benötigen, oder ihre Daten löschen lassen möchten, wenden sie sich bitte an: info@x5gon.org
